# The JERKS

### Dependencies

- Heroku CLI toolbelt
- Bundler (`gem install bundler`)

### Development

- Clone this repository `git clone git@github.com:kunalbhat/the-jerks.git`
- Install dependencies `bundle install`
- Run `ruby app.rb` to start the server
- Load `localhost:4567` in browser
